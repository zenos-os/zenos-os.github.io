Build process
  1. Build Zenos OS C# Components
  2. Build Native ZenosOS object files(ILCompiler)
  3. Build Assembly components
    • multiboot_header
    • boot
    • OS.obj
    • modules
    • dotnet
    • load_end_addr
  4. Link all object files
  5. Build ISO
